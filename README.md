# WebLLM SVG Generator

> [!Important]
> This project is written by AI Agent.

ブラウザ上で動作する AI 駆動の SVG ジェネレーター。WebLLM を使用して、完全にクライアントサイドで LLM を実行し、テキストプロンプトから SVG を生成します。

※今のところまともに生成できたことはありません＾＾

## 使い方

1. モデルを選択してロード（初回はダウンロードに時間がかかります）
2. プロンプト入力欄に SVG の説明を入力
3. 「生成」ボタンをクリック
4. 生成された SVG をプレビュー
5. 必要に応じてダウンロードまたは履歴から再利用
